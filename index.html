>

Use this tab when parents request a specific teacher.

>

CRM LeadsSales
New lead > Contact parent > Update status/notes > Set next follow-up
>

Add every new parent contact from ads, referral, walk-in, or WhatsApp.

>

Check Needs Action first. Follow the Next action shown for each lead.

Old sheet leads are inside Old Imported Leads. Expand only when checking old records.

Assessment / TrialSales / Admin
Lead agrees > Open booking tab > Book teacher slot > CRM updates
>

When CRM status changes to Assessment or Trial Class, choose to open the Assessment / Trial tab.

>

Choose a saved student or type the lead child's name if the child has not enrolled yet.

>

Fill date, 30-minute slot time, teacher, and subject. The session is treated as 25 minutes inside that slot.

>

If a matching CRM lead is found, confirm to update that lead with the booking details.

TeachersAdmin
Add teacher > Set subjects/rate/tier > Add slots > Save
>

Add teacher profile, subjects taught, tier, rate, and profile picture.

>

Add regular weekly slots or particular date slots.

>

Use bulk add for repeated slots such as Monday to Friday 6pm to 9pm.

StudentsAdmin
Find student > Set package/status > Add regular slots > Save
>

Search student name before adding, to avoid duplicates.

>

Choose package and mark status as registered or trial/not enrolled.

>

Add one or more regular class slots when the student attends more than once per week.

ReportsAdmin
Select month/year > Generate report > Check numbers > Export
>

Use CRM dashboard to check monthly leads and yearly comparison.

>

Use teacher report to check completed lessons, not show allowance, and payout estimate.

>

Export PDF or Excel when sending reference to management or teachers.

Action Guide

Salesperson Actions

>

New lead: Open CRM Leads > Create profile > Set New Contact > Set urgency/source > Set next follow-up.

>

Old imported lead replies: Open Old Imported Leads > Search lead > Move to Active > Update notes > Set next follow-up.

>

Parent replies: Edit lead > Update notes > Confirm child details > Update urgency > Set next follow-up.

>

Parent asks time: Open Weekly Timetable/Teacher Overview > Filter slots > Propose open slots only > Record options in notes.

>

Assessment: Set Assessment > Open Assessment / Trial > Type lead child name or choose student > Book assessment.

>

Trial: Set Trial Class > Open Assessment / Trial > Book trial > Update lead after trial date passes.

>

No reply: Use no-reply template > Set Follow Up or No Response > Set next follow-up date. If clearly closed, set Lost or Not Interested.

>

Enrollment: Set Enrolled > Open Students > Mark Registered > Add package and regular class slots.

Admin Actions

>

Teacher setup: Open Teachers > Add profile > Add subjects/tier/rate > Add timetable slots > Save.

>

Student regular slot: Open Students > Edit student > Add teacher/day/time/subject > Save.

>

Student not show: Click class > Set Student Not Show > Add remark > Save. It turns red and pays RM5 allowance.

>

Class cancelled: Click class > Choose Cancelled/Public Holiday/Teacher Leave > Add remark > Save. It turns grey and is not counted for payout.

>

Undo wrong cancellation: Open Replacement Classes > Find the case > Click Undo Cancellation > Original class returns to booked.

>

Teacher partial leave: Mark only affected class slots as Teacher Leave.

>

Teacher payout: Open Reports > Select teacher/month/year > Generate report > Export PDF.

>

Before important edits: Pull From Sheet > Edit > Save > Confirm auto-sync or Push manually.
